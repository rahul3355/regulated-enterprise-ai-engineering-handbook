# Support

## How to Get Help

This handbook is community-maintained and open-source. Here's how to get help:

### Questions About Content

- **Open an issue** using the [Question template](.github/ISSUE_TEMPLATE/question.md)
- **Start a discussion** in [GitHub Discussions](https://github.com/USERNAME/regulated-enterprise-ai-engineering-handbook/discussions)
- **Check the FAQ** in the [README](README.md#frequently-asked-questions)

### Technical Help

- **Code example not working?** Open a [Bug Report](.github/ISSUE_TEMPLATE/bug_report.md) with the file path and error details
- **Can't find what you need?** Open a [Feature Request](.github/ISSUE_TEMPLATE/feature_request.md) to suggest new content
- **Found an error?** Open a [Bug Report](.github/ISSUE_TEMPLATE/bug_report.md) or submit a PR with the fix

### Using This Handbook for Interviews

- Start with [interview-prep/](interview-prep/) for structured preparation
- Use [agents/interview-coach-agent.md](agents/interview-coach-agent.md) for AI-assisted mock interviews
- See [interview-prep/README.md](interview-prep/README.md) for the full preparation guide

### Using This Handbook for Projects

- Follow the [30/60/90 day plan](README.md#306090-day-study-plan)
- Use [skills/](skills/) for engineering checklists
- Reference [templates/](templates/) for design docs, RFCs, and runbooks
- Study [architecture/](architecture/) for decision record patterns

## What Support Looks Like

This is an **open-source documentation project**, not a product with SLAs. Support means:

- Community members helping community members
- Maintainers reviewing PRs and issues
- Content being improved based on feedback

It does **not** mean:

- Guaranteed response times
- Personal tutoring or mentoring
- Custom content on request
- Legal or compliance consulting

For those needs, we encourage you to **contribute** — the best way to get help is to help others.

## Enterprise Support

If you're an organization that wants to use this handbook internally and need:

- Custom content for your specific technology stack
- Training workshops for your engineering team
- Consulting on AI safety, security, or compliance

Please contact the maintainers directly to discuss arrangements.
